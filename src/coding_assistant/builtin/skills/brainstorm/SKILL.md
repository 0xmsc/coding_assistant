# Brainstorm

- Use this skill when the user wants options, sequencing advice, or tradeoff analysis.
- Do not edit files while brainstorming.
- Gather only the context needed to compare realistic approaches.
- Present a small number of concrete options with effort, risk, and reversibility.
- Recommend a next move and hand off to `plan` or `todo` once the scope is clear.
