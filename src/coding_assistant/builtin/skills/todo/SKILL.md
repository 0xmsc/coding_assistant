# TODO Tracking

- Keep a lightweight task list in `.agent/TODO.md` during execution work.
- Add concrete tasks before major edits and mark them complete as you finish them.
- Remove stale items so the list reflects only the current task.
- If implementation reveals new ambiguity or scope changes, return to `plan` or `brainstorm` instead of forcing ahead.
