# Planning

- Use this skill when the objective is clear but the work is large enough to benefit from a plan.
- Capture the objective, scope, constraints, milestones, and verification approach before implementation.
- Write plan files under `.agent/plans/` unless the user asked for another repo-local location.
- Do not edit implementation files while planning.
- Wait for explicit approval before handing execution off to `todo`.
